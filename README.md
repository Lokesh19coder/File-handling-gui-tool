#  Advanced File Handling Tool (GUI)

A Python-based GUI application built using Tkinter that performs file handling operations such as reading, modifying, and writing text files. The tool provides an interactive interface for efficient word replacement and text processing with built-in error handling.

---

##  Features

*  Browse and select text files
*  Find and replace specific words
*  Case-insensitive replacement option
*  Displays number of replacements made
*  Real-time output preview
*  Error handling (file not found, permission issues)
*  Simple dark mode interface
*  Clear output functionality

---

##  Technologies Used

* Python
* Tkinter (GUI Library)
* Regex (for advanced string manipulation)

---

##  Installation & Setup

### 1. Clone the repository

```bash
git clone https://github.com/your-username/file-handling-gui-tool.git
```

### 2. Navigate to project folder

```bash
cd file-handling-gui-tool
```

### 3. Run the application

```bash
python Filehandling.py
```

---

##  How to Use

1. Click **Browse** to select a `.txt` file
2. Enter the word to find
3. Enter the replacement word
4. (Optional) Enable **Ignore Case**
5. Click **Modify File**
6. View updated content in the output box
7. Click **Clear Output** if needed

---

##  Example

### Input File

```
Hello world
world is amazing
```

### Output After Replacement

```
Hello Earth
Earth is amazing
```
